# CMDB用户主机资源管理平台  

## 技术栈  

- Beego框架构建用户管理系统  
- HTML、CSS实现前端多功能展示  
- Bootstrap快速实现前端页面平台  
- web开发流程前后端结合  
- 用户权限管理和数据库开发  
- 实现阿里云、腾讯云接口调用管理  
- 插件设计模式  
- 设计开发多云主机的统一管理体系  
- Go实现多平台拓展体系架构  
- Prometheus API实战和核心组件源码解析  
- 针对使用consul进行服务发现及管理  
- 采集插件Exporter开发  
- Alertmanager webhook开发  
- webhook告警通知开发  
- K8s Go客户端使用以及服务的部署
